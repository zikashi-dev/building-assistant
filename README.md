# building-assistant
i am building an voice Assistant
Zikashi AI Assistant is a **personal voice-based AI assistant** designed to help users with daily tasks, smart control, and interactive conversation. The goal is to create a personal assistant that is:

- ✅ Fully voice-interactive (input & output)  
- ✅ Smart software control (open-ended AI interactions)  
- ✅ Lightweight and customizable for personal use  
- ✅ Easy to expand with new features  

This project is **open-source and experimental**, aiming to document the development process and showcase progress.
